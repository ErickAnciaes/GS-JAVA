# Orbiclean – Sistema de Monitoramento de Missões Espaciais

## Nome da Solução

**Orbiclean**

O Orbiclean é um sistema desenvolvido em Java para o cadastro, consulta e gerenciamento de missões espaciais voltadas ao monitoramento e mitigação do lixo espacial. O sistema permite registrar informações sobre missões, acompanhar seu status e gerar relatórios resumidos para auxiliar na tomada de decisões.

---

## Problema Escolhido

O aumento da quantidade de lixo espacial em órbita representa um dos principais desafios da exploração espacial moderna. Fragmentos de satélites desativados, estágios de foguetes e outros detritos podem causar colisões que colocam em risco satélites ativos, missões espaciais e futuras operações orbitais.

O sistema Orbiclean foi criado com o objetivo de simular o gerenciamento de missões espaciais responsáveis por monitorar, rastrear e propor soluções para a redução dos impactos causados pelo lixo espacial.

## Funcionalidades do Sistema

O sistema permite:

- Cadastro de missões espaciais;
- Listagem de missões cadastradas;
- Busca de missões por nome;
- Busca de missões por área de impacto;
- Alteração do status de uma missão;
- Exibição de resumo geral das missões;
- Identificação da missão com maior prioridade.

## Considerações Finais

O Orbiclean foi desenvolvido como uma simulação acadêmica para demonstrar a aplicação dos conceitos fundamentais de Java. O projeto busca conscientizar sobre os impactos do lixo espacial e a importância do desenvolvimento de tecnologias voltadas à sustentabilidade das atividades espaciais.

O sistema apresenta uma proposta alinhada aos Objetivos de Desenvolvimento Sustentável (ODS), demonstrando como a tecnologia pode contribuir para a preservação e segurança do ambiente orbital.
